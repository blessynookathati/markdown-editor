# Multi-Page React Markdown Editor

A feature-rich markdown editor built using:

- React + TypeScript
- React Router
- Zustand
- Tailwind CSS
- LocalStorage persistence
- Docker containerization

## Features

- Multi-page routing
- File management
- Live markdown preview
- Debounced auto-save
- Bold & Italic formatting
- Word counter
- Dark/Light theme
- Dockerized deployment

## Run Locally

npm install  
npm run dev  

## Run With Docker

docker-compose up --build
